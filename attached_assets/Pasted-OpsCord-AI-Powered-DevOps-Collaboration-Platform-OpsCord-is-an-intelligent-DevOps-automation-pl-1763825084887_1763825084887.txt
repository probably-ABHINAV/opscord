ğŸš€ Quick Start
Prerequisites
Node.js 18+ and pnpm
Supabase account
Google Gemini API key
GitHub OAuth app
Discord bot application
Installation
Clone and Install
pnpm install
Configure Environment Variables Create .env.local with:
# Supabase
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key

# Google Gemini AI
GEMINI_API_KEY=your_gemini_api_key

# Discord
DISCORD_CLIENT_ID=your_discord_client_id
DISCORD_TOKEN=your_discord_bot_token
DISCORD_PUBLIC_KEY=your_discord_public_key

# GitHub
GITHUB_OAUTH_ID=your_github_oauth_id
GITHUB_OAUTH_SECRET=your_github_oauth_secret

# App
NEXT_PUBLIC_APP_URL=http://localhost:5000
NEXTAUTH_SECRET=your_random_secret
Setup Database
# Run Prisma migrations
npx prisma db push
Register Discord Commands
curl -X POST http://localhost:5000/api/discord/register-commands
Start Development Server
pnpm run dev
Visit http://localhost:5000 to see OpsCord in action!

ğŸ—ï¸ Architecture
Tech Stack
Frontend: Next.js 15, React 19, TypeScript
Styling: Tailwind CSS with custom animations
Database: PostgreSQL (Supabase) + Prisma ORM
AI: Google Gemini 1.5 Flash
Auth: NextAuth.js with GitHub/Discord OAuth
APIs: GitHub REST API, Discord REST API
Key Workflows
PR Summarization Flow:

GitHub webhook triggers on PR open/update
Webhook stored in database
Background job queues AI summarization
Gemini analyzes PR and generates summary
Summary sent to Discord channel
Dashboard updated in real-time
Discord Command Flow:

User runs slash command in Discord
Discord sends interaction to /api/discord/interactions
Signature verified and command processed
Response sent back to Discord
